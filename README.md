# trying-susy

Just learning [Susy](http://susy.oddbird.net/). Basic setup with no Ruby dependency.

## Install

`npm i && bower i` to pull dependencies

# Usage

run `gulp` for development mode or `gulp --live` to build optimised version

## Features
- browser-sync
- uncss
- css autoprefixing
- live reload
- etc.
